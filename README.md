# AJSD-Debug-LWC-DIY

This repository contains a Lightning Web Component (LWC) with an associated static resource, used for practicing debugging techniques with Chrome DevTools.

## Prerequisites

- Salesforce CLI installed (`sf --version` to check)
- VS Code with Salesforce Extension Pack (recommended)
- Access to a Salesforce org (Dev/Sandbox/Trailhead Playground)

## Setup Steps

### 1. Clone the repository
git clone https://github.com/shobana-AJSD/AJSD-Debug-LWC-DIY.git
cd AJSD-Debug-LWC-DIY

### 2. Authenticate to your own Salesforce org
### 3. Deploy the Static resources to the org
### 4. Deploy the component
### 5. Place the component in Home Page
### 6. Start debugging
